This project analyzes Superstore sales data using Power BI to uncover insights related to sales performance, profitability, customer segments, and product categories.

The dashboard highlights:
- Overall sales, profit, and order trends over time
- Category and sub-category level performance analysis
- Region-wise and state-wise sales distribution
- Identification of high- and low-performing products
- Profitability comparison across customer segments and shipping modes

The objective of this dashboard is to support data-driven decision-making by transforming raw sales data into clear, interactive visual insights using Power BI.

Tools used:
- Power BI
